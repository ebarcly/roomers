# Roomers — Cozy Murder Logic

*Every room has a story. One hides a killer.*

A cozy murder-mystery logic puzzle for iPhone. Place every suspect, follow the
clues, and name the killer — whoever ends up alone in a room with the victim.
Every case has exactly one solution, provable by pure deduction. No guessing, ever.
